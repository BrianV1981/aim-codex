# Getting Started

This file is for the current `aim-codex` repo.

## Setup
```bash
cd /home/kingb/aim-codex
./setup.sh
source ~/.bashrc
```

The setup script installs the local virtual environment and registers the `aim-codex` alias.

## Initialize
```bash
aim-codex init
```

This scaffolds or refreshes the local workspace state and registers the Codex hook surface.
